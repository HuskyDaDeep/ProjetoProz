Métodos Utilizados

Esta página apresenta os principais métodos de invasão utilizados em ataques cibernéticos, explicando como funcionam, quais riscos oferecem e por que representam uma ameaça aos sistemas modernos.

Métodos de Invasão

Scanning (Varredura)

O scanning é utilizado na fase de reconhecimento do ataque, onde o invasor identifica portas abertas, serviços ativos e possíveis vulnerabilidades em um sistema.

Exemplo: um atacante descobre que um servidor possui portas abertas sem proteção adequada.

SQL Injection

A SQL Injection ocorre quando dados não são validados corretamente, permitindo a execução de comandos SQL maliciosos no banco de dados.

Exemplo: um invasor consegue acessar o sistema inserindo comandos SQL em um formulário de login.

XSS (Cross-Site Scripting)

O XSS permite a execução de scripts maliciosos no navegador de outros usuários, explorando falhas na validação de dados.

Exemplo: um comentário malicioso rouba cookies de sessão.

Engenharia Social

Explora o fator humano, utilizando manipulação psicológica para obter informações confidenciais.

Exemplo: um golpista se passa por suporte técnico e solicita a senha.

Força Bruta

Consiste em testar automaticamente diversas combinações de senha até encontrar a correta.

Exemplo: um script tenta milhares de senhas em um login.

Phishing

Golpe que utiliza mensagens falsas para enganar usuários e roubar dados sensíveis.

Exemplo: e-mail falso do banco pedindo confirmação de dados.

Ransomware

Malware que criptografa os dados da vítima e exige pagamento para liberação.

Exemplo: empresa perde acesso aos arquivos após abrir anexo malicioso.

Ataques a APIs

Exploram falhas de autenticação ou autorização em APIs mal configuradas.

Exemplo: acesso a dados de usuários apenas alterando o ID na URL.

Vulnerabilidades CVE

Falhas de segurança conhecidas e documentadas publicamente em softwares.

Exemplo: sistema desatualizado explorado por falha conhecida.

Exploits Automatizados

Utilizam scripts para explorar vulnerabilidades de forma rápida e em larga escala.

Exemplo: ataque automático em centenas de servidores.
